# Accessibilité – Conseils pour l’équipe

- Vérifiez le contraste des couleurs (ex : texte blanc sur fond bleu foncé).
- Testez la navigation au clavier (Tab, Entrée, etc.).
- Ajoutez des attributs `aria-label` sur les boutons importants si besoin.
- Privilégiez des tailles de texte lisibles (>14px).
- Utilisez des composants accessibles (ex : boutons natifs, balises <label> pour les champs).

> Pour aller plus loin, utilisez un outil comme [axe](https://www.deque.com/axe/) ou Lighthouse pour auditer l’accessibilité automatiquement.
