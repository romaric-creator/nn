# Sauvegarde et Restauration – Gestion Stock Boutique

## Sauvegarder la base de données
1. Fermez l’application Gestion Stock Boutique.
2. Rendez-vous dans le dossier d’installation de l’application.
3. Repérez le fichier de base de données (ex : `data.sqlite` ou `main.db`).
4. Copiez ce fichier sur une clé USB ou un autre support externe.

## Restaurer la base de données
1. Fermez l’application Gestion Stock Boutique.
2. Remplacez le fichier de base de données actuel par votre sauvegarde.
3. Rouvrez l’application.

**Astuce** : Effectuez une sauvegarde régulière, surtout avant toute mise à jour ou modification importante.

> Pour une automatisation, une future version pourra proposer un bouton « Sauvegarder » et « Restaurer » dans les paramètres.
