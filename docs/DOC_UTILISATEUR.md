# Documentation Utilisateur – Gestion Stock Boutique

Bienvenue dans l’application Gestion Stock Boutique !

## Fonctionnalités principales
- **Tableau de bord** : Vue d’ensemble de l’activité (produits, clients, ventes, alertes stock faible).
- **Gestion du stock** : Ajouter, modifier, consulter les produits et leur quantité.
- **Ventes** : Enregistrer une vente, consulter l’historique.
- **Clients** : Ajouter, modifier, consulter les clients.
- **Utilisateurs** : Gérer les comptes utilisateurs (accès sécurisé).
- **Rapports** : Générer des rapports sur les ventes et le stock.

## Actions rapides
- Depuis le tableau de bord, cliquez sur « Nouvelle Vente » ou « Gérer le Stock » pour accéder directement aux fonctions principales.

## Conseils d’utilisation
- Utilisez le menu à gauche pour naviguer.
- Les alertes de stock faible apparaissent sur la page d’accueil.
- Pensez à sauvegarder régulièrement vos données (voir section sauvegarde).

## Support
Pour toute question, contactez votre administrateur ou le support technique.
